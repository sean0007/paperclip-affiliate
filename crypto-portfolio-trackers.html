1. DIY Spreadsheet (Free)

★★★★☆ 4.3/5 · Price: $0
💰 Free

The simplest, cheapest option. A well-built Google Sheets or Excel spreadsheet gives you complete control over your data, formulas, and reporting. Takes 1-2 hours to set up but costs nothing and flexes to your needs.

What You Need

  • Google Sheets or Excel
  • CoinGecko API (free) for live prices
  • Basic spreadsheet knowledge
  • Manual entry for each trade (or CSV import from exchanges)
// Sample Google Sheets crypto tracker setup

COLUMN A: Date
COLUMN B: Exchange
COLUMN C: Pair (e.g., BTC/USDT)
COLUMN D: Side (Buy/Sell)
COLUMN E: Quantity
COLUMN F: Entry Price (USD)
COLUMN G: Cost Basis (=E*F)
COLUMN H: Current Price (from CoinGecko API)
COLUMN I: Current Value (=E*H)
COLUMN J: P&L (=I-G)
COLUMN K: P&L % (=(I-G)/G*100)

// CoinGecko price formula (Google Sheets)
=INDEX(GOOGLEFINANCE("CRYPTOCURRENCY:BTC-USD","price"),2,2)
// Or use IMPORTJSON for CoinGecko API:
=IMPORTJSON("https://api.coingecko.com/api/v3/simple/price?ids=bitcoin&vs_currencies=usd","bitcoin.usd")

Pros

  • Completely free
  • Complete control over data and formulas
  • Customizable to your exact needs
  • Can handle any portfolio complexity
  • No account needed

Cons

  • Manual entry required for each trade
  • No automated sync with exchanges
  • Tax reporting features need extra setup
  • API price fetching can break or rate-limit

Best For

Traders who want full control, have simple portfolios (under 20 positions), and don't need tax automation. Also great as a backup even if you use a paid tool.

★★★★★ 4.8/5 · Most Complete
💰 Free (limited) · $11.97–$29.95/month or $99–$299/year

The gold standard for crypto portfolio tracking. CoinTracking handles automated exchange imports, P&L calculation, tax reporting (US + 30+ countries), DeFi tracking, and more. It's what serious traders use.

Features

  • Auto-import from 100+ exchanges via API or CSV
  • Real-time P&L tracking with 20+ calculation methods
  • US tax reporting (same-day IRS Form 8949 exports)
  • DeFi / DEX tracking (PancakeSwap, Uniswap, etc.)
  • Gain/loss reports, capital gains summaries, income reports
  • Automated tax lot selection (FIFO, LIFO, HIFO, Specific ID)
  • Live trade alerts

Pros

  • Most complete feature set of any tracker
  • Supports 100+ exchanges automatically
  • Best-in-class tax reporting for US filers
  • Handles DeFi (airdrops, staking, LP) which breaks most trackers
  • 30+ countries' tax rules built in

Cons

  • Expensive for full features ($99/year minimum for what most need)
  • Interface feels dated vs newer competitors
  • Auto-import can miss some exchange transactions
  • Free tier is very limited (100 trades)

Pricing

  • Unofficial: $11.97/month — 300 trades, auto-import, basic tax
  • Hero: $29.95/month — unlimited, all features, priority support
  • Lifetime: $99–$299 one-time (best value if you trade a lot)

Best For

Active traders with multiple exchange accounts, anyone dealing with DeFi or staking, US-based traders who need tax reporting. Worth the cost if you're doing taxes on crypto gains.
